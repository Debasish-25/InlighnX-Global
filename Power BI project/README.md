# E-commerce Sales Dashboard (Power BI)
## Project Overview

This project involves creating an interactive E-commerce Sales Dashboard using Microsoft Power BI. The dashboard provides insights into sales, profit, quantity sold, and customer behavior, helping stakeholders make informed business decisions.

## Objective

Build a visually appealing and interactive dashboard

Track key sales and profit metrics

Analyze sales performance across states, categories, and customers

Enable dynamic filtering and drill-down analysis
