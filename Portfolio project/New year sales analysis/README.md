# New Year Sales Analysis
## Project Overview

This project focuses on analyzing New Year sales data to uncover meaningful insights about customer behavior, sales trends, and product performance. The analysis helps businesses make data-driven decisions for marketing and inventory planning.

## Objectives

Analyze overall sales performance during the New Year period

Understand customer demographics and purchasing behavior

Identify top-selling products and high-revenue categories

Detect trends and patterns using Exploratory Data Analysis (EDA)

## Dataset Information

File Name: New Year Sales Data.csv

Type: Structured CSV dataset

Contents include:

Customer demographics (age, gender, location)

Product categories

Order quantity and sales amount

Purchase trends during New Year

## Tools & Technologies Used

Python

Jupyter Notebook

Libraries:

pandas

numpy

matplotlib

seaborn

## Analysis Performed

Data cleaning and preprocessing

Handling missing and duplicate values

Exploratory Data Analysis (EDA)

Customer segmentation analysis

Sales trend visualization

Category-wise and product-wise revenue analysis

## Key Insights

Identified top revenue-generating product categories

Found customer groups contributing most to sales

Observed purchase trends during New Year season

Highlighted high-demand products
